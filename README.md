# Efficient Sorting and Searching System for Large Heterogeneous Datasets

This repository contains a flexible and efficient system designed to sort and search large datasets composed of heterogeneous data types, such as integers, strings, and composite keys. The system is built to adapt dynamically based on the nature and size of the dataset.

### Key Features

* **High Scalability:** Capable of handling dataset sizes ranging from thousands up to 10 million records.


* **Adaptive Sorting:** Utilizes Python’s built-in Timsort for its adaptability and O(n log n) average-case time complexity, making it ideal for nearly-sorted data.


* **Fast Searching:** Implements Binary Search for sorted datasets, ensuring optimal O(log n) performance.


* **Dynamic Operations:** Supports real-time insertions and deletions while minimizing performance degradation.



### Tech Stack

* **Language:** Python 3 


* **Libraries:** `dataclasses`, `random`, `time`
